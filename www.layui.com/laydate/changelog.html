    5.3.0 2021-05-08

    • [新增] isPreview 参数，用于控制是否显示当前选择值的预览（默认 true）
    • [优化] 日期范围选择，因左右日期面板独立，固取消范围区间标注，增加开始-结束文本预览
    • [优化] range 参数，可支持传入数组，用于分别指定开始日期和结束日期的选择器
    • [优化] 时间范围选择，初始结束时间为 23:59:59
    • [优化] 当前日期不在设定的最小（min）和最大（max）日期内，则自动校正面板可选的初始日期
    • [优化] 选中完毕的赋值逻辑，对非 input 元素，如果存在子元素，则不进行默认赋值操作，由回调去处理
    • [优化] 之前版本一直存在的初始处全局事件重复绑定的问题
    • [修复] 当开启范围选择，未触发 change 回调的问题
    • [修复] 在 ie11 因 laydate-day-mark 的 height：100% 导致的异常
  • 

    • [优化] link 核心方法

    5.2.0 2021-03-31

    • [优化] type 设置不规范时的问题，将自动纠正为默认 type
    • [优化] 弹出面板的坐标吸附算法，更加灵活合理
    • [修复] 弹出日期选择面板时，按回车键无效的问题
    • [修复] layDate 组件当 lang 设置为 'en' 时，部分提示内容未显示英文的问题
    • [剔除] 多余无效的 js 代码
  • 

    • [修复] 当设为年/月选择器时，点击选择年/月数值时，面板未自动关闭的问题
    • [修复] 因之前版本的范围联动方式造成的若干问题

    5.0.0

    2017-08-21
    • 全新重写
    • 支持单独显示年选择器、年月选择器、日期选择器、时间选择器、日期时间选择器
    • 支持双控件，用于选择年/年月/日期/时间/日期时间五种类型选择器的范围（可顺时、逆时）
    • 支持日期格式的自定义
    • 支持日期是否合法的自动校验
    • 支持有效日期范围的设定
    • 支持内置事件（可自定义）、外部事件、直接显示等多种调用方式
    • 支持中文版和国际版的语言设定
    • 支持开启公历节日和标记重要日期
    • 支持直接嵌套在页面的某个容器中
    • 支持底部按钮的任意顺序排版
    • 支持智能显示在最佳可视坐标
    • 支持回车快捷键选择
    • 支持多种内置主题的设定，支持自定义主题色，且可单独定制主题
